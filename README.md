proyecto FitLife
Información del proyecto:
Nombre: FitLife - Tu entrenador personal
Alcance:

Aplicación móvil para seguimiento de rutinas de ejercicio
Registro de progreso físico del usuario
Catálogo de ejercicios con descripciones e imágenes
Temporizador para entrenamientos
Historial de entrenamientos realizados

Requerimientos funcionales:

Crear y gestionar rutinas de ejercicio personalizadas
Visualizar lista de ejercicios por categoría (cardio, fuerza, flexibilidad)
Registrar entrenamientos completados con fecha y duración
Ver estadísticas de progreso semanal/mensual
Configurar perfil de usuario (peso, altura, objetivos)

Reglas de negocio:

Un entrenamiento debe tener al menos 3 ejercicios
La duración mínima de un ejercicio es 30 segundos
El usuario debe completar al menos 80% de una rutina para que cuente como finalizada
Las estadísticas se calculan solo con entrenamientos completados
No se pueden eliminar entrenamientos del historial (solo archivar)

Contexto:
Aplicación dirigida a personas que desean mantener una vida activa y saludable, permitiéndoles 
organizar sus entrenamientos de manera efectiva sin necesidad de un entrenador personal.
